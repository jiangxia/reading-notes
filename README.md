# 前言
任何一个知识点，如果不能成为自己已有知识网络的一部分，那它很快就会被我们的大脑当成垃圾回收了。

所以，最好的学习，是输出，在持续输出中，不断扩展、深化自己的知识体系。

此项目，将记录本人业余学习的各种知识，希望能逐渐形成自己的知识网络，也希望对你有所帮助~
